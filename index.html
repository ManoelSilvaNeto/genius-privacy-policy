Política de Privacidade

Última atualização: 19/08/2025

Este aplicativo é um jogo de passatempo do tipo Genius (Simon), desenvolvido para entretenimento.

Coleta de dados

  • Não coletamos, armazenamos ou compartilhamos dados pessoais (como nome, e-mail, telefone, documentos etc.).
  • O aplicativo não solicita login, cadastro ou permissões sensíveis.
  • Não utilizamos cookies nem tecnologias de rastreamento para identificar usuários.

Permissões do dispositivo

O jogo funciona sem exigir permissões além das necessárias ao seu funcionamento básico. Se no futuro alguma permissão for adicionada, esta política será atualizada para explicar a finalidade.

Publicidade e terceiros

Atualmente, o aplicativo não exibe anúncios e não integra SDKs de terceiros que coletem dados pessoais. Caso passemos a exibir anúncios no futuro, atualizaremos esta política com detalhes sobre parceiros e opções de consentimento.

Crianças

O jogo é adequado para todas as idades e não coleta dados pessoais de crianças.

Alterações nesta política

Podemos atualizar esta política periodicamente. Recomendamos revisar esta página para se manter informado.
